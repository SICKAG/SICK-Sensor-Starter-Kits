# IO-Link Starter Kit Overview
The IO-Link Starter Kit provides an introduction to working with IO-Link sensors. It includes hardware, documentation, and examples to help you get started quickly.

## Hardware
The IO-Link Starter Kit consists of the following parts:

- IO-Link Sensor  
- IO-Link Master  
- Power Supply  
- Network Cable  
- Network Adapter  
- Transport Box  

## Key Features
- Seamless IO-Link communication
- Simple configuration with SOPAS Air or compatible tools
- Easy integration into various automation systems
- Supports parameterization and diagnostics
- Plug-and-play setup
