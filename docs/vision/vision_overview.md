# Vision Starter Kit Overview
The Vision Starter Kit provides tools and examples to help you work with vision-based sensors. It includes sample code, configurations, and documentation to get started quickly.

![Vision Starter Kit](https://cdn.sick.com/media/pim/2/32/232/IM0089232.jpg)

## Hardware
The Vision Starter Kit contains of the following parts:

- **InspectorP61x**  
- **Mounting Plate** 
- **Power Supply**  
- **Network Cable**  
- **Network Adapter**  
- **Transport Box**  

## Key Features
- High-resolution image capture.
- Advanced object detection capabilities.
- Easy-to-use configuration tool (Nova2D)

Explore the [Getting Started Guide](./vision_getting_started.md) and advanced sections for more details.
